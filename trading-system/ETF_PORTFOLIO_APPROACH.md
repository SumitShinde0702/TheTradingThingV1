# ETF-Like Multi-Agent Portfolio System

## Concept

Instead of **consensus-based multi-agent** (agents vote on one decision), create an **ETF-like portfolio** where:
- **10 independent agents** trade separately
- Each agent has its own capital allocation (e.g., 10% each = 1000 USDT per agent)
- Each agent makes its own decisions independently
- You see **only the combined/aggregated P&L** (like an ETF)

## Pros & Cons

### โ Pros

1. **Diversification**
   - Different strategies reduce correlation
   - One bad agent doesn't kill the portfolio
   - Like holding 10 different stocks

2. **Risk Management**
   - Natural risk distribution
   - Can weight agents by performance
   - Easy to disable underperforming agents

3. **Performance Stability**
   - Less volatility than single agent
   - Smoother equity curve
   - Better Sharpe ratio (usually)

4. **Strategy Specialization**
   - Each agent can specialize (scalping, swing, trend)
   - Different timeframes, indicators, risk profiles
   - Market regime adaptation

5. **Scalability**
   - Easy to add/remove agents
   - Can allocate capital dynamically
   - Performance-based rebalancing

### โ Cons

1. **Capital Fragmentation**
   - 10 agents = 10x smaller positions per agent
   - May miss large opportunities
   - More positions to manage

2. **API Costs**
   - 10 agents = 10x API calls
   - More decision logs, database writes
   - Higher infrastructure costs

3. **Complexity**
   - More to monitor (though you only see aggregate)
   - More potential failure points
   - Harder to debug issues

4. **Over-Diversification**
   - Too many agents might cancel each other out
   - Diminishing returns after 5-7 agents
   - Optimal number is probably 5-10

### 2. Portfolio Aggregation

```go
type PortfolioManager struct {
    agents []*trader.AutoTrader
    totalCapital float64
}

func (pm *PortfolioManager) GetTotalEquity() float64 {
    total := 0.0
    for _, agent := range pm.agents {
        total += agent.GetEquity()
    }
    return total
}

func (pm *PortfolioManager) GetTotalPnL() float64 {
    return pm.GetTotalEquity() - pm.totalCapital
}
```
